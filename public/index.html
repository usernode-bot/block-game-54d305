Choose a mode to start.

🧱

Classic Mode

Build and explore with other players in a shared 3D world. Place and break any block you like — no timer, no pressure.

⏱️

Time Attack

Race the clock solo. Break as many blocks as you can — fresh arena every run, high score tracked.

Easy

Time Attack (60s)

Clear as many blocks as you can in 60 seconds. Blocks stack—no clearing allowed. Single-player only.

Daily Challenge

Same objective for all players. Resets every 24 hours. Compete on the leaderboard.

Endless Mode

Build forever on an endless grid. No timer, no score multiplier—just how long until you run out of valid moves?

🧩 Puzzle Mode

Clear a fixed set of blocks level by level. Solve increasingly difficult puzzles.

Tutorial

Learn the basics step-by-step: place blocks, break them, earn points, and clear lines.

Learn Mode

Perfect for getting started. Learn the basics at your own pace with helpful hints.

👁️

Watch

Fly through the world as a spectator. See what others are building, chat freely, and join any time.

    Controls

    • WASD / arrows — move
    • Space / Shift — up / down
    • Left-click or F — add a block
    • Right-click or G — delete a block
    • Ctrl+Z — undo last action (Classic only)
    • 1–9 — pick block type
    • H (playing) — hold / swap block type
    • H (menu) or (?) — toggle this panel
    • Enter — open / close chat
    • L or 🏆 — leaderboard
    • M or 🔊 — mute / unmute audio
    • Ctrl+Z — undo last action (Classic)
    • Ctrl+Y — redo (Classic)
    • Esc — close panel / release / menu
    • Power-ups — collect glowing cubes for Speed, Jump, or Rapid Place (hold F/Add)
