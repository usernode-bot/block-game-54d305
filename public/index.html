Controls

  • WASD / arrows — move
  • Space / Shift — up / down
  • Left-click or F — add a block
  • Right-click or G — delete a block
  • 1–9 — pick block type
  • H or (?) — toggle this panel
  • M or 🔊 — mute / unmute audio
  • Esc — release / menu
